# 🐇 Population Growth Calculator

A Python GUI application that uses **Linear Algebra** to model the long-term growth rate and distribution of a population (juveniles and adults) over time.

## 📌 Key Concepts Demonstrated
* **Eigenvalues:** Used to calculate the long-term growth rate of the population.
* **Eigenvectors:** Used to determine the stable ratio between juveniles and adults.
* **Population Matrices:** Mathematical representation of survival and reproduction rates.

## 🛠️ Technology Stack
* **Python 3**
* **Tkinter** (For the interactive Graphical User Interface)
* **NumPy** (For complex linear algebra calculations)
