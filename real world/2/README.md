# 🧮 Linear Algebra Real-World Applications

Welcome to my collection of Python applications! 🚀 
This repository contains interactive tools built to demonstrate how advanced mathematics (Linear Algebra) is used to solve real-world problems.

## 📌 Projects Included:

### 1. Population Growth Calculator 🐇 (Eigenvalues)
Models the long-term growth rate and distribution of a population (juveniles and adults) over time.
- **Concepts:** Eigenvalues, Eigenvectors, Population Matrices.
- **Tech Stack:** Python, Tkinter, NumPy.

### 2. Stock Market Risk Simulator 💸 (Cholesky Decomposition)
Simulates the risk and correlation between two company stocks to understand market behavior.
- **Concepts:** Covariance Matrices, Positive Definite Matrices, Cholesky Decomposition.
- **Tech Stack:** Python, Tkinter, NumPy.

### 3. Shape Deformation Visualizer 🎨 (Linear Transformations)
An interactive graphing tool that shows how a 2D physical object deforms under stress or transformation, plotting the exact stretch directions.
- **Concepts:** Linear Transformations, Eigenvectors (Stretch Direction), Eigenvalues (Stretch Amount).
- **Tech Stack:** Python, Matplotlib, NumPy.
