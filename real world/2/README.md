# 💸 Stock Market Risk Simulator

An interactive Python application that simulates the risk and correlation between two company stocks to understand market behavior using **Cholesky Decomposition**.

## 📌 Key Concepts Demonstrated
* **Covariance Matrices:** Represents the variance and correlation between two stocks.
* **Positive Definite Matrices:** Ensures the mathematical validity of the risk model.
* **Cholesky Decomposition:** Breaks down the matrix into a lower triangular matrix ($L$) to simulate correlated market variables.

## 🛠️ Technology Stack
* **Python 3**
* **Tkinter** (For the interactive Graphical User Interface)
* **NumPy** (For matrix decompositions and linear algebra)
